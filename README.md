# Arch Suckless
My Suckless setup


## Installing

```
cd {dwm}
sudo make install
cd ..
cd {slstatus}
sudo make install
```

## Pre-reqs

Arch Based Systems (Manjaro, Arch, ArchLabs etc)

```sudo pacman -S base-devel```

Debian Based Systems (Debian and Ubuntu based distros)

```sudo apt-get install build-essentials xorg-dev```

## Shortcut Keys

`Alt + p` - dmenu

`Alt + Shift + Return/Enter` - Terminal
